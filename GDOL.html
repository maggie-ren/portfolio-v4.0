Georgia Department of Labor(GDOL) is an administrative agency of the U.S. state of Georgia. During the pandemic, this website’s view surged because this is the website many Georgians applied for unemployment benefits.

Timeline and contribution

Two week project. Team Size 4. A proto-persona was created as a team, along with the usability test. I was personally incharge of the user interface design.

Role

UI Designer

Why Redesign GDOL

Our team decided on GDOL website because during the pandemic, large communities of Georgians become unemployed and needed to apply for unemployment benefits at GDOL. Since the website was so poorly put together, it made getting access to the much needed benefits unnecessarily harder. We wanted to help remedy by redesigning and improving the GDOL website entirely.

Usability Test Matrix

After creating our proto-persona, we did usability tests and took test notes, used 2*2 matrix to group feedback and voted on them then decided which features are more important to develop.

Navigation Analysis

We performed an analysis of the navigation of the website, along with testing and getting feedback from users. The overall conclusion is that the website is confusing.

New Navigation Sitemap

As a first step we performed card sorting on the navigation. From the results, we regrouped and remapped the entire navigation system for GDOL. This new navigation sitemap will simplify the experience compared to the original labyrinth of a navigation.

Iterations

Iteration, iteration, iteration! 3 rounds of feedback and improvement was done with users to create a better experience every time.

UI Style Guide

I kept orange as the main color because it is Georgia’s state color but changed the complementing color to improve the color contrast and appeal.

Digital Wireframes

After our improvements, the website as a whole is more user friendly and intuitive.
